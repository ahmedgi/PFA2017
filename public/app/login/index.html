Scholar Manager

Scholar Manager est une application qui permet de génerer une filière selon les accréditations du ministère et aussi les charges des professeurs dans un département, et il fournit des statistiques ... il offre en plus l'export des notes de APPOGEE vers une nouvelle application.

Télécharger

Linux

Pour pouvoir tester l'application il faut suivre les étapes suivantes :

ouvrir un terminal

installer mongodb en tapant les commandes suivantes : 

$ echo "deb http://repo.mongodb.org/apt/debian wheezy/mongodb-org/3.2 main" | sudo tee /etc/apt/sources.list.d/mongodb-org-3.2.list
$ sudo apt-get update
$ sudo apt-get install -y mongodb-org

installer curl par la commande suivante:

$ sudo apt-get install curl

installer nodejs par les commande suivantes:

$ curl -sL https://deb.nodesource.com/setup_6.x | sudo -E bash -
$ sudo apt-get install -y nodejs

installer bower : 

$ npm install -g bower

installer git : 

$ sudo apt-get install git

cloner le projet : 

$ git clone https://github.com/zakariarhb/PFA2017.git

entrer dans la repertoire du projet : 

$ cd PFA2017

installer les dependances coté serveur : 

$ npm install

creér une repertoire data : 

$ mkdir data

lancer la base de donner mongodb : 

$ mongod -db path data

lancer un autre terminal et deplacer dans la repertoire du projet PFA2017 :

puis déplacer dans la repertoire public : 

$ cd public

installer les dépendances front-end : 

$ bower install

revenir au repertoire courant PFA2017 et lancer l'application : 

$ node app.js

l'url de l'application est : localhost:8010

Windows

Pour pouvoir tester l'application sur Windows, il faut suivre les étapes suivantes :

Etape 1 : Installation des outils nécessaires

Pour pouvoir démarrer un serveur nodejs Rendu-vous à la page de téléchargement de nodejs Cliquer sur Windows Installer pour une installation aisé, sinon Vous avez le choix du type d'installation.

Pour lancer le serveur SGBD orienté document (noSQL) MongoDB , consulter la documentation de mongoDB

Installez git pour pouvoir utilser bower par la suite.

Pour installez bower Ouvrir un terminal git bash et tapez :

$ npm install -g bower

Etape 2 : Clonage du projet et installation des dépendances

Ouvrir un terminal git bash et taper 

$ cd "chemin vers votre espace de travail"

Ensuite, cloner le projet : 

$ git clone https://github.com/zakariarhb/PFA2017.git

Entrer dans la repertoire du projet : 

$ cd PFA2017

Installer les dependances côté serveur : 

$ npm install

Creér une repertoire data : 

$ mkdir data

Lancer la base de donner mongodb en cmd admin: 

"C:\Program Files\MongoDB\Server\3.2\bin\mongod.exe" --dbpath=data -storageEngine=mmapv1

Remplacer le chemin où mongodb est installé, lancer un autre terminal et aller au répertoire du projet PFA2017 :

Puis aller au repertoire public : 

$ cd public

Installer les dépendances front-end : 

$ bower install

Revenir au repertoire PFA2017 et lancer l'application : 

$ node app.js

l'url de l'application est : localhost:8010

Et tester l'application, et si vous rencontrez des problèmes n'hésitez pas à nous contacter.
